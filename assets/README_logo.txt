Support + Instagram buttons:
- Set Render environment variable BMC_URL to your BuyMeACoffee link (e.g. https://buymeacoffee.com/yourname)
- Optional: set IG_URL (defaults to https://instagram.com/redlightsoff5)

If BMC_URL is not set, the Support button will show disabled.
